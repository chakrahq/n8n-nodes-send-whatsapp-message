# @chakrahq/n8n-nodes-send-whatsapp-message

An n8n community node for sending WhatsApp messages via the Chakra API. This node enables you to send both session messages and template messages directly from your n8n workflows.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/sustainable-use-license/) workflow automation platform.

## Features

- **Session Messages**: Send text messages directly to WhatsApp users
- **Template Messages**: Send pre-approved WhatsApp template messages with dynamic parameters, images, videos, documents, and location data

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Credentials

This node requires Chakra WhatsApp API credentials. Configure your credentials in n8n using the `chakraWhatsappApi` credential type.

For more information, refer to the [Chakra API documentation](https://apidocs.chakrahq.com).
